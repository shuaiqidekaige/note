## 语句
语句会执行一些操作但是不会返回一个值。

## 表达式
- 表达式会在求值后返回一个值。
- 表达式不能包含分号，不然会变成语句。
- 表达式如果不返回任何值，会隐式地返回一个 ()

## 函数
- 声明函数的关键字 fn
- 函数名和变量名使用[蛇形命名法(snake case)](https://course.rs/practice/naming.html)，例如 `fn add_two() -> {}`
- 函数也是表达式.
- 函数必须指明参数类型。
- 默认隐式返回单元类型（）
- 当用 `!` 作函数返回类型的时候，表示该函数永不返回任何值( diverge function ), 发散函数。
```Rust
fn dead_end() -> ! { 
  panic!("你已经到了穷途末路，崩溃吧！"); 
}
```
